# @a2amesh/internal-adapter-base

Abstract `BaseAdapter` (extends `A2AServer`) and shared contract helpers for provider-specific A2A adapters.

Consumed by `@a2amesh/runtime-adapter-*` packages. Not intended for direct end-user consumption.

## Exports

| Export path  | Content                                                            |
| ------------ | ------------------------------------------------------------------ |
| `.`          | `BaseAdapter` abstract class                                       |
| `./contract` | `createTextArtifact`, `extractText`, `extractRequiredText` helpers |

See [Compatibility](../../docs/compatibility.md) for supported Node.js and package ranges.
